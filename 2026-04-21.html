1 ruvnet/RuView

将普通 WiFi 信号转化为空间智能传感平台。通过 ESP32-S3 捕获 Channel State Information (CSI)，实现：

  • 穿透墙体感知：检测人、呼吸、心率，无需摄像头
  • 17 关键点姿态估计：基于 CMU DensePose From WiFi 研究
  • 多节点网状协同：4-6 个 ESP32-S3 节点覆盖 360° 空间
  • 边缘智能：65 个 WASM 模块直接在 ESP32 上运行

技术亮点：Rust v2 版本达到 810x 加速、54K fps 管道；呼吸检测 6-30 BPM、心率 40-120 BPM、存在感知 100% 准确率；模型仅 8KB（4-bit 量化）可嵌入 ESP32 SRAM。

2 santifer/career-ops

  • ATS 优化 PDF 生成：Keyword-injected CVs
  • 门户扫描：预配置 45+ 公司（Anthropic、OpenAI、ElevenLabs 等）
  • 批量处理：并行评估 10+ offer

技术栈：Claude Code + Node.js + Playwright + Go (Bubble Tea TUI)。适合求职者用 AI 对抗 AI 筛选。

3 titanwings/colleague-skill

将离职同事的数字遗产转化为可复用的 AI Skill。输入飞书消息、钉钉文档、Slack 聊天记录等，输出能模拟该同事工作风格和人格的 AI Skill。

架构：双层结构（Work Skill + Persona）。Persona 采用 5 层模型：hard rules → identity → expression → decisions → interpersonal。支持版本控制和增量更新。

趋势分析

核心关键词：WiFi CSI 传感、DensePose、ESP32 边缘智能、Skill Distillation、Humanoid Robot

技术连贯性

  • WiFi 传感（RuView）和机器人控制（北京半马机器人）代表两条不同路径的空间智能：前者用无线信号穿透感知环境，后者用物理形体执行任务。两者都在摆脱对摄像头/显示器的依赖。
  • Skill Distillation（colleague-skill）从"知识蒸馏"延伸到"人格蒸馏"，预示 AI 技能将从工具属性向人格属性演进。

风向转变

  • Humanoid Robot 进入实用化阶段：北京半马 50 分 26 秒完赛，机器人不再只是 demo，开始挑战真实世界任务。
  • 免费 LLM API 竞争加剧：Cerebras/Groq 的 ultra-fast inference、OpenRouter 的多样性，正在压缩商业 LLM API 的利润空间。

行动建议

  1. 关注 RuView 类 WiFi 传感技术：隐私敏感场景（医疗、监控）的低成本替代方案，可能是下一个增长点
  2. 跟踪 OpenRouter/Groq 免费 tier 演进：速率限制逐步放宽，免费 API 正成为开发者默认选择
  3. Humanoid Robot 供应链：机器人本体厂商已跑通，但核心零部件（灵巧手、关节电机、传感器）国产化机会值得关注
