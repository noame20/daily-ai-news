GitHub 项目

1 jackwener/opencli

Make Any Website & Tool Your CLI — 一个通用 CLI Hub 和 AI 原生运行时，可将任意网站、Electron 应用或本地二进制工具转换为标准化命令行界面，专为 AI Agent 通过统一 `AGENT.md` 集成来发现、学习和执行工具而构建。

核心技术亮点：

维度 说明
AI Agent 集成 提供 `opencli-operate` skill，Claude Code / Cursor 等 Agent 可直接控制浏览器
浏览器复用 重用 Chrome/Chromium 已登录状态，凭证不离开浏览器
反检测机制 内置反指纹措施，绕过 `navigator.webdriver` 检测
零 LLM 成本 运行时无需消耗 token
73+ 内置适配器 覆盖 Bilibili、知乎、小红书、Twitter、Reddit、YouTube 等主流平台

技术栈： TypeScript (ESM) + Node.js/Bun + VitePress 文档 + vitest 测试

核心命令：

  • `opencli doctor` — 诊断连接状态
  • `opencli explore` — 发现网站 API
  • `opencli operate` — 浏览器自动化控制
  • `opencli register` — 注册外部 CLI

安装： `npm install -g @jackwener/opencli` | 版本 1.6.1 | Apache-2.0

风向判断

  • 利好： opencli 的零 LLM 成本 + 反检测 + 多平台适配器组合，降低了 AI Agent 实际落地门槛
  • 信号： 从单一浏览器自动化扩展到"网站+应用+CLI"的统一抽象层，Agent 工具生态正走向标准化协议层（AGENT.md）
