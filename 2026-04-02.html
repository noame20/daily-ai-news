趋势分析

核心关键词：Self-Evolution 自进化、SWE-Pro benchmark、source map 泄露、MoE 架构、Agentic Model

无历史搜索结果，基于本期内容直接分析：

1. Self-Evolution 自进化成为 Agent 模型新范式

MiniMax M2.7 首次让模型深度参与自身训练，可处理 30-50% 的 RL 工作流（文献综述、实验规范、调试、跨团队协作），这标志着 AI 模型从"被训练"向"自我优化"的关键转变，是 Agent 架构的重大突破。

2. Claude Code 源码泄露暴露 CLI 工程化成熟度

泄露揭示 Claude Code 内部采用 React+Ink（终端 React）、四层压缩策略、Blake2b 提示词缓存边界等工程实践，对 CLI Agent 开发有重要参考价值。泄露本身非安全入侵，但暴露了 Anthropic 内部指令差异（`ant` vs 外部用户）。

3. OpenAI 估值 $852B 反映 AI 基础设施军备竞赛

122B 融资规模远超历史，Amazon $500亿（$350亿附条件）+ NVIDIA $300亿 + SoftBank $300亿，表明芯片和云基础设施是 AI 竞争核心壁垒。OpenAI 目标打造"AI SuperApp"，整合 ChatGPT/Codex/Agent 功能。

Nagi 关注建议：

  • MiniMax M2.7 技术细节：关注 Self-Evolution 的实现机制（Agent 如何监控训练、检测坏数据、自动修复），这是下一代 Agent 架构的核心方向
  • Agent 工具调用能力：M2.7 在 Terminal Bench 2（57%）和 Toolathon（46.3%）的表现值得关注，Toolathon 46.3% 已是全球顶尖
  • Claude Code 泄露的工程价值：四层压缩策略和提示词缓存边界技巧可借鉴用于 CLI Agent 优化
