[1] daytonaio/daytona

Daytona 是一个安全且弹性的 AI 代码执行基础设施，为 AI 生成的代码提供隔离的沙箱运行环境。其核心亮点是亚 90 毫秒的沙箱创建速度，从代码到执行环境即刻完成。项目支持 Python、TypeScript、Ruby、Go、Java 等多语言 SDK，兼容 OCI/Docker 镜像，可实现代码的 Git 操作、LSP 协议、程序化执行等 API 控制。沙箱可永久持久化存储，支持大规模并发 AI 工作流。架构采用多服务分离设计：API（NestJS）、CLI（Go）、Runner（Go）、Dashboard（React）等组件协同工作，通过 Nix flakes 实现可复现的开发环境。

[2] juspay/hyperswitch

Hyperswitch 号称"Linux for Payments"，是一个用 Rust 编写的开源支付路由交换机，让支付更快速、更可靠、更低成本。其核心思想是将支付逻辑解耦，通过智能路由将每笔交易导向成功率最高的支付服务商（如 Stripe、Adyen、PayPal 等 40+ 连接器）。项目采用 Rust 开发（高性能、无 unsafe 代码），PostgreSQL + Redis 作为数据层，配套 OpenTelemetry + Prometheus + Grafana 监控体系。提供收入恢复、成本洞察、文件存储、智能路由、对账自动化等完整支付模块，支持 ACH、SEPA、Apple Pay、Klarna 等多种支付方式。部署支持 Docker、本地化或云端（AWS/GCP/Azure）。

[3] Nagi-ovo/gemini-voyager

Gemini Voyager 是 Google Gemini 和 AI Studio 的全能增强套件，以浏览器插件形式（支持 Chrome、Firefox、Edge、Safari）提供十余项实用功能：时间轴导航（可视节点跳转对话历史）、文件夹管理（双层级+颜色标记）、提示词库（跨 Gemini/AI Studio 共用）、聊天导出（JSON/Markdown/PDF）、Google Drive 云同步、Deep Research 链接提取、Mermaid 流程图渲染、ChatGPT 风格的视觉特效等。技术栈采用 React 19 + Tailwind CSS + Vite 构建，支持 10 种语言国际化，Manifest V3 服务工作线程架构。借助 Gemini Voyager，用户可以更高效地管理和追溯与 AI 的对话内容。

[4] rowboatlabs/rowboat

Rowboat 是一个开源 AI 同事助手，通过将工作内容转化为知识图谱来帮助你完成各种任务。它连接邮箱和会议记录，构建持久化的记忆图谱，在此基础上完成撰写文档/幻灯片（如"帮我做一个关于Q3路线图的演示"）、会议准备（汇总过往决策和待办）、追踪人物/公司/话题的动态更新、甚至语音备忘录等任务。核心差异在于本地优先设计：所有数据以纯 Markdown 格式存储（Obsidian 兼容），知识图谱存于本地 Qdrant 向量数据库，支持 Ollama 本地模型或自有 API key 的远程模型。技术栈为 Electron 桌面应用 + React UI + Vercel AI SDK，可扩展接入 Model Context Protocol (MCP) 和 Composio 工具库。

[5] forrestchang/andrej-karpathy-skills

这是一个基于 Andrej Karpathy 关于 LLM 编程陷阱观察总结的 Claude Code 行为准则插件（本质是一个 CLAUDE.md 文件），旨在改善 LLM 编码时常犯的错误行为。四大核心原则：编码前先思考（不假设、不隐藏困惑，主动暴露权衡）、简洁优先（最少代码解决问题，不做投机性代码）、精准修改（只触碰必须改动的，不"改进"无关代码）、目标驱动执行（将任务转化为可验证目标，先写测试再实现）。安装方式灵活：可通过 Claude Code 插件系统一键安装，也可直接下载 CLAUDE.md 到任何项目。该项目简单但深刻，直击 LLM 在编程任务中的常见认知偏差。

[6] OpenBMB/VoxCPM

VoxCPM 是多语言语音合成模型，最新版本 VoxCPM2（2B 参数）实现了真正的 Tokenizer-Free 架构——无需离散化标记，直接通过扩散自回归混合生成连续语音表示。核心流程：Local Encoder → TSLM（MiniCPM-4 骨干语言模型）→ RALM（残差声学语言模型）→ Local DiT（基于流匹配的去噪扩散解码器）。亮点功能：支持30 种语言的 TTS、语音设计（通过文本描述控制性别/年龄/情感/语速）、可控语音克隆、真实感声音延续；输出质量达 48kHz 录音棚级别（内置超分辨率），RTF 实时因子约 0.3。提供 Python SDK、CLI 工具、Web Demo，以及 LoRA 微调支持。生态包括 Nano-vLLM 高吞吐 GPU 推理、VoxCPM.cpp CPU 推理、ComfyUI 集成等。
