### 4. deepagents - LangChain 全功能 Agent Harness

**Agent harness built with LangChain and LangGraph. Equipped with a planning tool, a filesystem backend, and the ability to spawn subagents.**

- 开箱即用的 Agent，无需自己接线 prompts 和工具
- 内置 planning (`write_todos`)、filesystem、shell、sub-agents 工具

- 大输出自动保存到文件

**快速开始**：
```python

```

### 6. workerd - Cloudflare JS/Wasm 运行时

**The JavaScript / Wasm runtime that powers Cloudflare Workers.**

- Cloudflare Workers 背后的开源运行时
- 用途：自托管 Workers 应用、本地开发测试、可编程 HTTP 代理
- **设计原则**：Server-first、Standard-based、Nanoservices、Homogeneous deployment、Capability bindings、永久向后兼容
- **Nanoservices**：类似微服务但性能如本地函数调用
- **安全**：Capability 绑定免疫 SSRF 攻击

**警告**：workerd 本身不是硬化的沙箱，运行不可信代码时需在安全沙箱（如虚拟机）中运行

**支持平台**：Linux、macOS（x86-64 和 arm64）、Windows（x86-64）

[GitHub](https://github.com/cloudflare/workerd)
