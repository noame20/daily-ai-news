[2] webadderall/Recordly

开源跨平台屏幕录制工具，支持 Mac/Windows/Linux。录制后可添加自动缩放区域、光标动画、点击效果、气泡式摄像头叠加、文本标注等功能，并支持导出为 MP4 或 GIF。使用 Electron + PixiJS 技术栈，各平台使用原生捕获 API（macOS 用 ScreenCaptureKit，Windows 用 WGC）。

[3] emdash-cms/emdash

基于 Astro 和 Cloudflare 的全栈 TypeScript CMS，是 WordPress 的精神继承者。亮点包括：沙箱化插件安全（插件运行在隔离的 Worker 中）、Portable Text 存储格式、内置 MCP 服务器支持 AI 交互、支持从 WordPress 导入内容。采用 Kysely 做 SQL 抽象层，支持 D1/SQLite/Turso/PostgreSQL 等多种数据库。

[4] motiful/cc-gateway

位于 Claude Code 和 Anthropic API 之间的反向代理，用于规范化设备指纹和遥测数据，保护用户隐私。核心功能包括：全身份重写（device_id、email、session metadata）、40+ 环境维度替换、账单头剥离（可降低约85%系统提示词成本）、集中式 OAuth 管理、代理感知支持。零登录客户端设置，部署后客户端机器从不联系 platform.claude.com。

[5] multica-ai/multica

开源平台，将编码 Agent 转变为真正的团队成员。支持任务分配、领取、执行、状态更新，Agent 会主动报告阻塞问题。技术栈：Next.js 前端 + Go 后端 + PostgreSQL（pgvector），通过 WebSocket 实时流式传输进度。Agent 解决的问题可转化为可复用技能供团队共享。支持 Claude Code 和 Codex 两种运行模式。
