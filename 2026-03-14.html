📊 GitHub 热门项目

microsoft/BitNet

官方 1 位 LLM 推理框架 - 微软推出的 bitnet.cpp 是 1.58 位大语言模型的官方推理框架，支持在 CPU 和 GPU 上进行快速无损推理。在 ARM CPU 上实现 1.37x-5.07x 加速，能耗降低 55.4%-70.0%；在 x86 CPU 上实现 2.37x-6.17x 加速，能耗降低 71.9%-82.2%。甚至可以单 CPU 运行 100B 参数模型，速度达到人类阅读水平（5-7 tokens/秒）。
🔗 https://github.com/microsoft/BitNet

langflow-ai/openrag

一站式 RAG 平台 - 基于 Langflow、Docling 和 Opensearch 构建的综合检索增强生成平台，整合了文档处理、向量检索和生成流程。
🔗 https://github.com/langflow-ai/openrag

lightpanda-io/browser

为 AI 设计的无头浏览器 - Lightpanda 是专为 AI 和自动化打造的无头浏览器，优化了 AI 代理与网页交互的场景。
🔗 https://github.com/lightpanda-io/browser

🔗 https://github.com/promptfoo/promptfoo

Siri AI 延期拖累苹果智能家居

苹果的智能家居显示屏再次延期，原因是新 Siri AI 助手尚未完工。该项目原计划 2025 年春季发布，后推迟至今年 3 月，现在仍需等待。该设备旨在成为家庭 AI 中枢，可显示个性化日历、提醒和笔记等数据。
🔗 https://tldr.tech/tech/2026-03-10

Claude Code 推出代码审查功能

Anthropic 为 Claude Code 推出 Code Review 功能，在每次 PR 上调度多个代理团队来捕捉 bug。该功能专为深度审查而非速度设计，目前面向团队和企业用户提供研究预览版。Anthropic 几乎对所有 PR 都运行 Code Review，帮助审查员覆盖遗漏的问题。
🔗 https://tldr.tech/tech/2026-03-10

GPT-5.4 发布 🤖

OpenAI 发布 GPT-5.4，提供 gpt-5.4 和 gpt-5.4-pro 两个新 API 模型，同时上线 ChatGPT 和 Codex CLI。知识截止日期为 2025 年 8 月 31 日，支持 100 万 token 上下文窗口。重点提升了创建和编辑电子表格、演示文稿和文档的能力，在所有相关基准测试中超越 GPT-5.3-Codex。
