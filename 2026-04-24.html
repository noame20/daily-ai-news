1 forrestchang/andrej-karpathy-skills

源自 Andrej Karpathy 关于 LLM 编码陷阱的观察——模型常做错误假设、不管理自己的困惑、不寻求澄清、不呈现权衡取舍、代码过度复杂化。该项目将 Karpathy 的洞见凝练为四大原则写入 CLAUDE.md：

  • Think Before Coding：不假想，显式陈述假设，困惑时停下来问
  • Simplicity First：最少代码解决问题，不做投机性抽象
  • Surgical Changes：只改必要的，改完自己清理孤立的 import/variables
  • Goal-Driven Execution：把指令转化为可验证目标，用测试循环验证

支持 Claude Code 插件安装、Cursor 规则文件，以及合并到现有项目 CLAUDE.md。已获 80.3k stars。

2 KeygraphHQ/shannon

Shannon Lite——白盒自主 AI 渗透测试工具，面向 Web 应用和 API。通过分析源代码识别攻击向量，然后执行真实漏洞利用（注入、认证绕过、SSRF、XSS），只报告有可工作的 PoEx 的漏洞，全程无需人工干预。

核心流程五阶段：Pre-Recon → Recon → Vulnerability Analysis（并行）→ Exploitation（"No Exploit, No Report"原则）→ Reporting。在 XBOW benchmark 上达到 96.15%（100/104 exploits）。覆盖 OWASP Juice Shop、crAPI 等靶场。AGPL-3.0 开源，Pro 版为商业闭源。
⭐Stars 🍴Forks 🔀RR 🔖Issue

3 vercel-labs/skills

开源 Agent Skills 生态的 CLI 工具 `npx skills`，让各 Coding Agent（Claude Code、Cursor、Codex、OpenCode 等 41+ Agent）共享同一套可复用的 SKILL.md 指令集。

核心功能：安装/移除/更新/搜索 Skills，支持 GitHub shorthand、完整 URL、本地路径、Git URL 等多种来源格式；安装范围分 Project（`.//skills/`）和 Global（`~/.config//skills/`）两种；Symlink 方式保持单一真相源。Skills 规范遵循 agentskills.io，可在任意 Agent 间迁移。`skills init` 命令可创建 SKILL.md 模板。
⭐Stars 🍴Forks 🔀RR 🔖Issue

4 Alishahryar1/free-claude-code

轻量级代理，将 Claude Code 的 Anthropic API 调用路由到免费/低成本替代 LLM 提供商，实现零成本使用 Claude Code。

支持的 Providers：

  • NVIDIA NIM：每分钟 40 请求免费
  • OpenRouter：数百种模型
  • DeepSeek：直接 API
  • LM Studio：完全本地
  • llama.cpp：本地 + Anthropic 兼容端点

仅需 2 个环境变量即可切换，无需修改 Claude Code 本身。支持 Per-Model Mapping（可将 Opus/Sonnet/Haiku 路由到不同模型）和 Thinking Token 解析。MIT License，Python 3.14 + uv 包管理。

5 nesquena/hermes-webui

Hermes Agent 的 Web UI 界面，无需构建步骤、无框架、无打包器，Python + 原生 JS 实现。暗黑主题三栏布局（左侧会话导航 + 中央聊天 + 右侧工作区文件浏览），Composer Footer 包含模型/Profile/工作区控制，上下文环直观显示 Token 使用量。

支持 SSE 流式响应、多提供商模型（OpenAI/Anthropic/Google/DeepSeek/OpenRouter 等）、会话管理、文件编辑、语音输入、7 种内置主题、密码认证、Slash 命令。Docker 支持单/双/三容器部署。测试覆盖率极高（1898 测试/53 文件）。

6 YouMind-OpenLab/awesome-gpt-image-2

GPT Image 2 精选提示词库，共收录 1526 条创意提示，涵盖头像、社交媒体、信息图、YouTube 缩略图、漫画、产品营销、电商主图、游戏资产、海报等多个场景，支持摄影、动漫、3D 渲染、像素艺术、水彩、赛博朋克等数十种风格。
