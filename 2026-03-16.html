1. claudian - Obsidian 的 Claude Code 插件

⭐ +109 | 4,017

将 Claude Code 嵌入 Obsidian 知识库，支持完整 Agent 能力。

核心功能：

  • 文件读写/搜索/bash 命令/多步工作流
  • 上下文感知：自动附加当前笔记、`@` 文件
  • 视觉支持：拖拽/粘贴图片分析
  • 内联编辑：word-level diff 预览
  • Slash 命令、Skills 扩展、自定义 Agent
  • MCP 支持：连接外部工具
  • 计划模式：Shift+Tab 先设计后实施

要求：Claude Code CLI + Obsidian v1.8.9+

2. vite-plus - Web 开发统一工具链

⭐ +622 | 2,096

整合 Vite/Vitest/Oxlint/Rolldown 等于一体。

核心命令：

  • `vp env` - 管理 Node.js
  • `vp install` - 安装依赖
  • `vp dev` - 开发服务器，瞬间 HMR
  • `vp check` - 格式化/lint/类型检查
  • `vp test` - 运行测试
  • `vp build` - 生产构建
  • `vp run` - 单体仓库任务

安装：`curl -fsSL https://vite.plus | bash`
