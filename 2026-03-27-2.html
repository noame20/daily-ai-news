1. hacksider/Deep-Live-Cam

只需一张照片即可实现实时换脸和视频深伪的AI工具。支持 webcam 实时换脸、多脸同时映射、嘴巴蒙版保留原始口型动作。兼容 CUDA/CoreML/DirectML/OpenVINO 等多种推理后端，覆盖 Windows/Linux/macOS（含 Apple Silicon）。内置审核机制拦截不当内容。项目被 Ars Technica、CNN Brasil、Bloomberg 等媒体报道。

2. thedotmack/claude-mem

Claude Code 插件，自动捕获编码会话中的所有操作并用 AI 压缩，在未来会话中注入相关上下文。核心基于 SQLite 持久存储 + Chroma 向量数据库实现语义搜索，提供 `mem-search` skill 支持自然语言查询历史。提供 Web UI（localhost:37777）实时查看记忆流，支持 `` 标签保护敏感内容，零配置安装。

3. EverMind-AI/MSA

Memory Sparse Attention——可扩展的端到端潜记忆框架，将检索和生成整合为单一可微分循环，在 2×A800 GPU 上实现 100M token 吞吐量。近 O(L) 训练和推理复杂度，KV cache 压缩 + Memory Parallel 推理引擎。MS MARCO 测评平均 3.760 分，超越 RAG +16%、Rerank +11.5%、HippoRAG2 +14.8%。论文已发表于 arXiv。
⭐Stars 2,242 🍴Forks 123 🔀PR 1 🔖Issue 3

4. open-gitagent/gitagent

框架无关的 Git 原生 AI Agent 定义标准。通过 `agent.yaml` + `SOUL.md` / `DUTIES.md` / `RULES.md` 等文件声明式定义 agent 的身份、技能、工具、工作流和合规规则。支持 Claude Code/OpenAI/CrewAI/Gemini 等多平台导出适配器，内置职责分离（SOD）合规审计引擎。提供 minimal/standard/full 三种模板，2行即可创建一个 hello world agent。
⭐Stars 1,866 🍴Forks 196 🔀PR 2 🔖Issue 12

5. hectorvent/floci

命名取自"floccus"（看起来像爆米花的云）——永远免费、开源的 AWS 本地模拟器。LocalStack 社区版 2026 年 3 月停产后，Floci 成为无 Token 限制的替代方案。启动仅需 ~24ms，内存占用 ~13 MiB，Docker 镜像约 90 MB。21+ AWS 服务支持，408/408 SDK 测试全部通过，含 API Gateway v2、Cognito、ElastiCache、RDS（IAM auth）等 LocalStack 社区版缺失的功能。
⭐Stars 1,901 🍴Forks 75 🔀PR 16 🔖Issue 30

6. letta-ai/claude-subconscious

为 Claude Code 打造的后台"潜意识"agent。基于 Letta 的记忆系统和上下文工程能力，在每个会话结束后分析 transcript，主动在下一轮对话前"耳语"提示。可读取代码库、搜索网络、跨项目共享记忆。完全不写 CLAUDE.md，所有内容通过 stdout 注入。默认 `whisper` 模式仅推送消息；`full` 模式注入完整记忆块。
