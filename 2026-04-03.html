趋势分析

核心关键词：MCP Server、Multi-Provider Agent、群体智能引擎（Swarm Intelligence）、Agent Harness、TDD开发流程

技术演进关系

1. Agent 基础设施：从单 Agent 到"AI 公司"的升级路径清晰
3月15日 Superpowers（Skills 框架，+59% Stars）和今日 Paperclip（多 Agent 编排成公司）呈现出连续演进：Superpowers 解决"一个人如何用 Agent 正确开发软件"，Paperclip 解决"如何让一群 Agent 像公司一样协作"。oh-my-openagent 则代表"Harness 层的军备竞赛"——Hash-anchored 编辑将 Agent 可靠性推向新高度。

2. MCP Server 已成为 Agent 标配基础设施
f/prompts.chat 内置 MCP Server、opencode 支持 MCP 协议，oh-my-openagent 内置 Exa/Context7/Grep.app 三大 MCP。MCP 不再是可选插件，而是 Agent 工具链的核心传输层。

3. 模型绑定 vs 模型无关：开源 Agent 的路线之争
opencode 坚持多模型路由（provider-agnostic），而 oh-my-openagent 通过 Sisyphus 实现自动模型选择（最佳模型完成最佳任务）。两条路线正在收敛——最终目标都是消除人工模型选择负担。

风向信号

  • Superpowers 爆发式增长（83k → 132k，2.5 周 +59%）：Skills 驱动的开发方法论被市场强烈认可，TDD + 子 Agent 驱动开发正在成为 Agent 软件工程的事实标准。
  • OpenAI 二级市场遇冷 vs Anthropic Cowork Agent 预期爆火：投资者情绪正在从"押注 OpenAI"转向"分散配置 Anthropic/其他"。Anthropic 明确表示 Cowork Agent（面向更广泛用户群）潜力大于 Claude Code（开发者工具），意味着 Agent 大众化浪潮临近。

预测与行动建议

接下来两周（2026-04-03 → 2026-04-17）预测：

  1. 更多 MCP Server 集成到主流 Agent 工具（Prompt 库、代码编辑器、数据源）
  2. Superpowers 类 Skills 框架将出现更多模仿者和竞争者
  3. Anthropic Cowork Agent 正式发布后，可能引发"企业 Agent"赛道新一轮融资热潮
  4. oh-my-openagent 的 Hash-anchored 编辑方案可能被其他 Harness 借鉴
