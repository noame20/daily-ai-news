1 luongnv89/claude-howto

Claude Code 权威指南项目，涵盖从基础到高级的完整学习路径。项目包含 10 个学习模块，包括：斜杠命令（Slash Commands）、持久化记忆（Memory）、技能（Skills）、子代理（Subagents）、MCP 协议配置、钩子（Hooks）、插件（Plugins）、检查点（Checkpoints）以及高级功能。

亮点：

  • 提供可直接 copy-paste 的模板，覆盖 Claude Code 开发的各个环节
  • MIT 许可证，v2.3.0 版本（2026年4月），兼容 Claude Code 2.1+
  • 学习路径分三级：入门（~3小时）→ 进阶（~5小时）→ 高级（~5小时）
  • 重点模板示例：
    • Code Review Skill：安全分析 + 性能审查 + SOLID 原则检查
    • Subagent 模板：专业代码审查子代理，可自动委托任务
    • Hook 示例（format-code.sh）：工具调用后自动格式化代码（Prettier/Black/gofmt）
    • MCP 配置：GitHub、Filesystem 等服务器的标准化 JSON 配置
  • 支持中文（zh/）和越南语（vi/）翻译
