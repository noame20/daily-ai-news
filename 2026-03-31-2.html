2. zai-org/GLM-OCR

GLM-OCR：智谱开源的多功能 OCR 工具，主打"精准 × 快速 × 全面"。在 OmniDocBench V1.5 上达到 94.62 分，排名全球第一。仅 0.9B 参数却能处理复杂表格、代码文档、印章等难例。 支持 vLLM、SGLang、Ollama 等推理框架，提供 Python SDK 和 CLI 工具，可快速完成文档解析。还支持 LoRA 微调（8GB 显存起），技术报告发表于 arXiv (2603.10910)。

3. TheTom/turboquant_plus

TurboQuant Plus：ICLR 2026 论文的实现，专注于 LLM 推理的 KV Cache 压缩。通过 PolarQuant + QJL（Walsh-Hadamard 旋转）算法，实现 3.8-6.4 倍压缩率，同时 perplexity 损失极小（turbo4 模式下仅 +0.23%）。亮点是 Sparse V 优化：在 32K 上下文下解码速度提升 22.8%。已在 llama.cpp 中实现 C/Metal GPU 内核，兼容 Apple Silicon 和多款 NVIDIA/AMD 显卡。提供 511+ 单元测试，100% 覆盖率。

4. 34306/vphone-aio
