[1] ronitsingh10/FineTune

FineTune 是一款 macOS 菜单栏应用，可为每个应用独立控制音量、支持多设备输出、音频路由和10频段EQ均衡器。完全开源免费，是 SoundSource 的替代品。

亮点功能：

  • 单应用音量控制：每个应用独立音量滑块，支持2x/3x/4x增益
  • 多设备音频路由：可将不同应用路由到不同输出设备
  • 10频段EQ + AutoEQ耳机校正：内置20种预设，支持搜索数千个耳机配置文件
  • 蓝牙设备管理：可直接从菜单栏连接配对设备
  • 通过URL Scheme自动化：支持从终端、Shortcuts、Raycast或脚本调用

[2] JCodesMore/ai-website-cloner-template

一个开源模板，通过 AI 编码 agents 对任何网站进行像素级完美克隆。只需指向 URL，运行 `/clone-website` 命令即可。

工作流程：

  1. 侦察阶段：截图、提取设计标记、交互扫描
  2. 基础阶段：更新字体颜色、下载所有资源
  3. 组件规范：编写精确到计算后 CSS 值的详细规范文件
  4. 并行构建：在 git worktree 中调度 builder agents 并行构建各组件
  5. 组装与QA：合并 worktree、视觉差异对比

支持 14+ 种 AI coding agents：Claude Code、Codex CLI、OpenCode、Cursor、Windsurf 等。

[3] SakanaAI/AI-Scientist-v2

AI Scientist-v2：通过 Agentic Tree Search 实现 Workshop 级别自动化科学发现。这是首个完全由 AI 生成并通过同行评审的 Workshop 论文系统。

核心功能：

  • 全自动科学研究流程：自动生成假设→运行实验→分析数据→撰写论文
  • 渐进式 Agentic Tree Search：实验管理器 Agent 引导的树搜索算法
  • 多模型支持：OpenAI GPT 系列、Claude (via AWS Bedrock)、Google Gemini
  • 无模板依赖：相比 v1 移除了人工编写模板，跨 ML 领域泛化能力更强

注意：v2 采用探索式方法，适合开放性科学探索；v1 模板驱动，适合明确目标。

  • 12 种 Prompt 模板：RTF、CO-STAR、RISEN、CRISPE、Chain of Thought、Few-Shot 等
