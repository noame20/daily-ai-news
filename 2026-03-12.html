🔥 今日精选

GitHub 项目

microsoft/BitNet - 1-bit LLM 推理框架

微软官方发布的 1-bit 大语言模型推理框架，实现1.37x-5.07x的 CPU 速度提升，能耗降低55%-70%。最新优化支持并行内核，可再提升1.15x-2.1x速度。能在单个 CPU 上运行 100B 参数的 BitNet 模型，速度达到人类阅读水平（5-7 tokens/秒）。

🔗 https://github.com/microsoft/BitNet

666ghj/MiroFish - 群体智能预测引擎

盛大集团出品的群体智能引擎，通过提取现实世界的种子信息（新闻、政策、金融信号），构建高保真平行数字世界。成千上万个具备独立人格的智能体在其中自由交互演化，可推演未来走向——让未来在数字沙盘中预演。支持上传数据分析报告或小说故事，返回详细预测报告。

🔗 https://github.com/666ghj/MiroFish

google-ai-edge/LiteRT - 谷歌边缘 AI 框架

TensorFlow Lite 的继任者，Google 官方发布的设备端高性能 ML 和 GenAI 部署框架。新增Compiled Model API，支持自动化加速器选择、真正的异步执行、高效 I/O 缓冲处理。统一 NPU 加速，支持 Google Tensor、Qualcomm、MediaTek 等芯片。提供最佳的 GPU 性能和卓越的生成式 AI 推理能力。

🔗 https://github.com/google-ai-edge/LiteRT

InsForge/InsForge - Agent 全栈后端平台

专为 AI 编码 Agent 设计的后端开发平台。通过语义层暴露数据库、认证、存储、函数等后端原语，让 Agent 可以理解、推理和操作后端系统。支持获取后端上下文、配置原语、检查后端状态。获 Vercel OSS Program 支持。

🔗 https://github.com/InsForge/InsForge

vectorize-io/hindsight - Agent 记忆系统

专为让 Agent学习而非仅仅记忆设计的记忆系统。在 LongMemEval 基准测试中达到 SOTA 性能，超越 RAG 和知识图谱等替代技术。已被 Fortune 500 企业和 AI 初创公司用于生产环境。支持 LLM Wrapper 快速集成，只需 2 行代码即可为现有 Agent 添加记忆能力。

🔗 https://github.com/vectorize-io/hindsight

alibaba/page-agent - 网页 GUI Agent

阿里巴巴开源的页内 JavaScript GUI Agent，无需浏览器扩展/Python/无头浏览器，仅需一行脚本即可在网页中集成。支持基于文本的 DOM 操作、自定义 LLM、人机交互 UI。适用于 SaaS AI Copilot、智能表单填充、无障碍访问等场景。

🔗 https://github.com/alibaba/page-agent

obra/superpowers - Agent 技能开发框架

完整的 Agent 软件开发工作流，基于可组合的"技能"系统。特色功能：

  • 启动后先与用户对话明确需求，而非直接写代码
  • 分块展示设计规格供用户审阅
  • 生成清晰的实现计划，强调 TDD、YAGNI、DRY 原则
  • 启动子代理驱动开发流程，可自主工作数小时
    支持 Claude Code、Cursor 等平台的官方插件市场。

🔗 https://github.com/obra/superpowers

langflow-ai/openrag - 智能 RAG 平台

基于 Langflow、Docling、OpenSearch 构建的综合 RAG 平台，支持智能文档搜索和 AI 驱动的对话。特色：

  • 预打包即用
  • Agentic RAG 工作流，支持重排序和多 Agent 协调
  • 智能文档解析，处理复杂真实数据
  • 拖拽式工作流构建器
  • 企业级搜索能力

🔗 https://github.com/langflow-ai/openrag

fishaudio/fish-speech - SOTA 开源 TTS

Fish Audio S2 版本的语音合成系统，支持 expressive voice cloning。提供 HuggingFace 模型、技术报告、Docker 镜像。遵守 FISH AUDIO RESEARCH LICENSE。

🔗 https://github.com/fishaudio/fish-speech

💡 趋势与信号

今日核心主题

今天的 GitHub Trending 呈现出清晰的Agent 生态爆发信号：

  1. Agent 基础设施完善 - 从后端（InsForge）、记忆（Hindsight）、技能（Superpowers）到前端交互（Page-Agent），Agent 开发的全链路工具链正在快速成熟

  2. 边缘 AI 成为主流 - BitNet（微软）和 LiteRT（谷歌）两大巨头同时发力 1-bit 模型和设备端推理，标志着 AI 从云端向边缘转移的关键转折点

  3. 群体智能实用化 - MiroFish 将群体智能从学术研究推向实际应用，支持政策预演、舆情推演等场景，反映了 AI 从单点能力向系统级模拟的演进

历史对比

检索历史记忆发现，关于"Agent 记忆"的讨论早在 2 月底就已出现（用户配置 Agent 记忆系统的咨询），但当时缺乏成熟方案。今天 Hindsight 的热门（2752 stars）说明市场需求已得到响应。

而 1-bit LLM 和边缘 AI 部署是全新的热点，尚未在历史简报中出现，这可能是下一个技术周期的起点。

趋势判断

  • 短期（1-3 个月）：Agent 开发工具将继续爆发，预计会出现更多垂直领域的专用框架
  • 中期（3-6 个月）：边缘 AI 设备将支持本地运行更大模型，可能催生新的应用场景
  • 长期（6-12 个月）：群体智能可能成为决策辅助的标准工具，特别是在政策制定、市场预测等领域
